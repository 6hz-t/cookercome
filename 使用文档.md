## 1. 平台简介

**CookerCome** 是一个连接客户与厨师的 O2O 服务平台，提供上门烹饪服务预约、订单管理、评价系统等完整功能。

- **用户端**：预约厨师上门服务，享受私人定制烹饪体验
- **厨师端**：展示个人技能，接收订单，提供上门服务
- **管理端**：平台运营管理，厨师资质审核，订单监管

## 3. 用户端使用指南

### 3.1 注册与登录

#### 注册账号
1. 访问用户端地址：http://124.221.15.13:81
2. 点击「注册」按钮
3. 填写手机号、验证码等信息完成注册

#### 登录
1. 输入手机号和密码
2. 点击「登录」即可进入平台

### 3.2 首页浏览

登录后进入首页，您可以：
- 查看平台推荐的优质厨师
- 浏览热门服务套餐
- 快速搜索附近厨师

### 3.3 查找厨师

#### 厨师列表
1. 点击「厨师」或「找厨师」入口
2. 查看附近厨师列表（按距离排序）
3. 支持筛选条件：
   - **菜系**：川菜、粤菜、湘菜等
   - **价格区间**：根据预算筛选
   - **评分**：按用户评价排序

#### 分页浏览
- 每页默认显示 10 条厨师信息
- 可通过页码切换查看更多厨师
- 支持调整每页显示数量（5/10/20/50 条）

#### 搜索与筛选
- **按菜系搜索**：输入擅长菜系关键词
- **按等级筛选**：初级/中级/高级/名厨
- **重置筛选**：点击「重置」恢复默认列表

#### 厨师详情
点击任意厨师卡片查看详情：
- 厨师简介、从业年限
- 擅长菜系和招牌菜品
- 服务价格和范围
- 用户评价记录

### 3.4 预约下单

1. **选择厨师**：在厨师详情页点击「预约」
2. **填写信息**：
   - 服务时间
   - 服务地址（支持地图选点）
   - 用餐人数
   - 特殊要求
3. **确认订单**：核对服务信息
4. **支付订单**：完成支付后等待厨师接单

### 3.5 订单管理

进入「我的」→「我的订单」：

| 订单状态 | 说明 | 可操作 |
|----------|------|--------|
| 待接单 | 已提交，等待厨师接单 | 取消订单 |
| 已接单 | 厨师已确认 | 联系厨师 |
| 服务中 | 厨师正在服务 | - |
| 待评价 | 服务完成 | 评价厨师 |
| 已完成 | 已评价 | 查看评价 |
| 已取消 | 订单取消 | - |

### 3.6 地址管理

进入「我的」→「地址管理」：

1. **新增地址**：填写联系人、电话、详细地址
2. **地图选点**：支持地图定位自动填充地址
3. **设为默认**：可设置一个默认地址
4. **编辑/删除**：管理已有地址

### 3.7 个人中心

进入「我的」→「个人中心」：

- **个人信息**：修改头像、昵称
- **我的收藏**：查看收藏的厨师
- **设置**：修改密码、退出登录

## 4. 厨师端使用指南

### 4.1 注册与登录

#### 注册账号
1. 访问厨师端地址：http://124.221.15.13:80
2. 点击「注册」按钮
3. 填写手机号、验证码等信息完成注册

#### 登录
1. 输入手机号和密码
2. 点击「登录」进入厨师端

### 4.2 资质认证

> ⚠️ **重要**：新注册厨师需完成资质认证才能接单

1. 进入「我的」→「资质认证」
2. 提交以下材料：
   - 身份证正反面照片
   - 健康证照片
   - 厨师证/职业资格证书（如有）
   - 个人简介
3. 等待平台审核（1-3 个工作日）
4. 审核通过后即可开始接单

### 4.3 信息管理

#### 个人资料
进入「我的」→「个人资料」：
- 修改头像、昵称
- 填写从业年限、擅长菜系
- 设置服务价格和半径

#### 服务状态
- **可接单**：正常接收订单
- **休息中**：暂停接收订单

### 4.4 菜品管理

进入「菜品管理」：

1. **添加菜品**：
   - 上传菜品图片
   - 填写菜品名称、描述
   - 设置价格
2. **编辑/删除**：管理已有菜品
3. **设置招牌菜**：标记主打菜品

### 4.5 订单处理

进入「订单管理」：

| 订单状态 | 说明 | 可操作 |
|----------|------|--------|
| 待接单 | 有新订单 | 接单/拒单 |
| 已接单 | 已确认订单 | 联系客户 |
| 待服务 | 等待上门服务 | 开始服务 |
| 服务中 | 正在服务 | 完成服务 |
| 待评价 | 服务完成 | 查看评价 |
| 已完成 | 已完成 | - |

#### 接单流程
1. 查看订单详情（时间、地址、人数、要求）
2. 点击「接单」确认
3. 按时上门服务
4. 服务完成后点击「完成服务」

#### 拒单说明
如无法接单，请选择拒单并填写原因：
- 时间冲突
- 超出服务范围
- 其他原因

### 4.6 收入统计

进入「我的」→「收入统计」：

- **今日收入**：当日完成订单收入
- **本周收入**：本周累计收入
- **本月收入**：本月累计收入
- **收入明细**：查看每笔订单收入

### 4.7 消息中心

进入「消息」：

- **订单通知**：新订单、订单状态变更
- **系统通知**：平台公告、活动信息
- **客户消息**：与客户的沟通记录

## 5. 常见问题

### 5.1 用户端常见问题

**Q: 如何联系厨师？**
A: 订单被接单后，可在订单详情页查看厨师联系方式或直接在线沟通。

**Q: 可以取消订单吗？**
A: 在厨师接单前可随时取消；接单后取消需与厨师协商。

**Q: 服务不满意怎么办？**
A: 可在评价中如实反馈，平台会跟进处理；如有严重问题请联系客服。

**Q: 如何保证食品安全？**
A: 平台所有厨师均需提供健康证并通过资质审核，建议查看厨师评价选择。

### 5.2 厨师端常见问题

**Q: 资质审核需要多久？**
A: 一般 1-3 个工作日，审核结果会通过消息通知。

**Q: 如何设置服务价格？**
A: 进入「个人资料」→「服务定价」，可设置基础服务费和菜品价格。

**Q: 可以拒单吗？**
A: 可以，但频繁拒单可能影响接单优先级。

**Q: 收入如何结算？**
A: 订单完成后收入自动计入账户，具体提现规则请咨询平台客服。
